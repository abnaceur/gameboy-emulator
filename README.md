# GMBU PROJECT

### Description

This project aims to reproduce the game boy / gameboy color emulator on the web.

### Technologies
- Angular v10.0.14
- Electron v10.1.0
- Electron Builder v22.8.0


### Build for developement
- First install dependencies
```bash
npm i
```

- Start the Electron/web server
```bash
npm start
```
